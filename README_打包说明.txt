凡人修仙3w天 - 快速打包说明
==========================================

重要提示：
本项目使用 Flet 框架，不是 Kivy！
不要使用 buildozer，请使用 Flet 自带的打包工具。

==========================================
快速开始：
==========================================

1. 安装 Flet
   pip install flet

2. 进入项目目录
   cd D:\desktop\mortal_3W_day\mortal_3W_day_1

3. 打包 Android APK
   flet build apk --module main_mobile

4. 等待打包完成
   生成的 APK 文件位于: build/apk/app-release.apk

==========================================
如果遇到闪退问题：
==========================================

1. 查看详细打包指南：
   打包指南.md

2. 使用移动端优化版：
   已创建 main_mobile.py，包含错误处理
